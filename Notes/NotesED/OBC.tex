\subsection{Open Boundary Condition Case}

When $g_I=0$, we have $g_F=0$, and the fermionic model has open boundary conditions (OBC). In this case, translational invariance is broken, and we cannot apply the Fourier transformation used for periodic systems. Instead, we use the Bogoliubov-de Gennes (BdG) formalism and the analytic diagonalization approach developed by Lieb, Schultz, and Mattis.

Starting from the fermionic Hamiltonian with open boundary conditions:
\begin{align}
    H_{\mathrm{spin}}^{\mathrm{OBC}} &= \frac{\cos\theta}{2} \sum_{n=1}^{N-1} (a_n a_{n+1} + a_n a_{n+1}^\dag - a_n^\dag a_{n+1} - a_n^\dag a_{n+1}^\dag) \notag\\
                                     &\quad -\frac{\sin\theta}{2} \sum_{n=1}^N (a_n a_n^\dag - a_n^\dag a_n).
\end{align}
We can rewrite this Hamiltonian in matrix form using the Nambu spinor formalism. For the one-body terms $a_n^\dag a_m$, we define a matrix $A$ with elements:
\begin{align}
    A_{nm} &= \sin\theta \delta_{nm} - \frac{\cos\theta}{2}(\delta_{n,m+1} + \delta_{n+1,m})
\end{align}
Similarly, for the pairing terms $a_n a_m$ and $a_n^\dag a_m^\dag$, we define an antisymmetric matrix $B$ with elements:
\begin{align}
    B_{nm} &= \frac{\cos\theta}{2}(\delta_{n,m+1} - \delta_{n+1,m})
\end{align}
Using the $2N$-component Nambu spinor $\bm{\Psi} = (a_1, \dots, a_N, a_1^\dag, \dots, a_N^\dag)^T$, the Hamiltonian takes the compact form:
\begin{align}
    H_{\mathrm{spin}}^{\mathrm{OBC}}
     &= \frac{1}{2} \bm{\Psi}^\dag H_{\mathrm{BdG}} \bm{\Psi}
\end{align}
where $H_{\mathrm{BdG}}$ is the $2N \times 2N$ Bogoliubov-de Gennes Hamiltonian matrix:
\begin{align}
    H_{\mathrm{BdG}} &=
    \begin{pmatrix}
        A  &B\\
        -B &-A
    \end{pmatrix}
\end{align}
To diagonalize $H_{\mathrm{spin}}^{\mathrm{OBC}}$, we use a Bogoliubov transformation to find new fermionic operators $\gamma_k, \gamma_k^\dag$ that are linear combinations of $a_n, a_n^\dag$:
\begin{align}
    \gamma_k      &= \sum_{n=1}^N (u_{nk} a_n + v_{nk} a_n^\dag)\\
    \gamma_k^\dag &= \sum_{n=1}^N (u_{nk}^* a_n^\dag + v_{nk}^* a_n)
\end{align}
With these operators $\gamma_k$ the Hamiltonian should become diagonal, and the eigenvalues should be the mode energies $\epsilon_k$:
\begin{equation}
    H = \sum_k \epsilon_k \left(\gamma_k^\dagger \gamma_k - \frac{1}{2}\right).
\end{equation}
which should give the commutation relations:
\begin{equation}
    [H, \gamma_k] = -\epsilon_k \gamma_k.
\end{equation}
such that we can derive the Bogoliubov-de Gennes eigenvalue equations:
\begin{align}
    A u_k + B v_k  &= \epsilon_k u_k \label{eq:BdG_eq1}\\
    -B u_k - A v_k &= \epsilon_k v_k \label{eq:BdG_eq2}
\end{align}
where $u_k = (u_{1k}, \dots, u_{Nk})^T$ and $v_k = (v_{1k}, \dots, v_{Nk})^T$ are $N$-component vectors containing the coefficients of the transformation.

Following Lieb, Schultz, and Mattis, we introduce new vectors $\phi_k = u_k + v_k$ and $\psi_k = u_k - v_k$. Adding and subtracting the BdG equations \cref{eq:BdG_eq1,eq:BdG_eq2}, we get:
\begin{align}
    (A - B) \psi_k &= \epsilon_k \phi_k \label{eq:LSM_eq1}\\
    (A + B) \phi_k &= \epsilon_k \psi_k \label{eq:LSM_eq2}
\end{align}
Eliminating $\psi_k$ (or $\phi_k$), we obtain:
\begin{align}
    (A - B)(A + B) \phi_k &= \epsilon_k^2 \phi_k \label{eq:E2_phi}\\
    (A + B)(A - B) \psi_k &= \epsilon_k^2 \psi_k \label{eq:E2_psi}
\end{align}
The key insight from the Lieb-Schultz-Mattis approach is that for our specific matrices $A$ and $B$, the products $(A+B)(A-B)$ and $(A-B)(A+B)$ have tridiagonal form. With the convention above,
\begin{align}
    A+B &= \sin\theta\,\Id - \cos\theta\,U,\\
    A-B &= \sin\theta\,\Id - \cos\theta\,L,
\end{align}
where $L_{n,n-1}=1$ and $U_{n,n+1}=1$ are the lower and upper shift matrices on the open chain. Thus
\begin{align}
    (A+B)(A-B) &= \sin^2\theta\,\Id
        - \sin\theta\cos\theta\,(L+U)
        + \cos^2\theta\,UL,\\
    (A-B)(A+B) &= \sin^2\theta\,\Id
        - \sin\theta\cos\theta\,(L+U)
        + \cos^2\theta\,LU.
\end{align}
In the bulk this gives the difference equation
\begin{equation}
    -\sin\theta\cos\theta\,(\chi_{n-1}+\chi_{n+1})
    + \chi_n = \epsilon^2 \chi_n.
\end{equation}
This tridiagonal structure allows for an analytic solution. The eigenvectors of these matrices have the form:
\begin{align}
    \phi_k &= \mathcal{N}_k (\sin N\omega_k, \sin (N-1)\omega_k, \ldots, \sin\omega_k)^T\\
    \psi_k &= \mathcal{N}_k (\sin\omega_k, \sin 2\omega_k, \ldots, \sin N \omega_k)^T
\end{align}
where $\mathcal{N}_k$ is a normalization constant, and the values of $\omega_k$ are determined by the transcendental equation:
\begin{align}
    \frac{\sin(N+1)\omega_k}{\sin N\omega_k} = \cot\theta
\end{align}
The eigenvalues (squared energies) for these eigenvectors are:
\begin{align}
    \epsilon_k^2 = 1 - \sin(2\theta)\cos\omega_k
\end{align}
giving us the energy eigenvalues:
\begin{align}
    E_k = \epsilon_k = \sqrt{1 - \sin(2\theta)\cos\omega_k}
\end{align}
The diagonalized Hamiltonian takes the form:
\begin{align}
    H_{\mathrm{spin}}^{\mathrm{OBC}} &= \sum_{k=1}^N \epsilon_k \left(\gamma_k^\dag \gamma_k - \frac{1}{2}\right)
\end{align}

\subsection{The numerical method}

There is a more direct and numerical method. If we write in the basis
\begin{equation}
    \bm{\Psi}' = (a_1, a_1^\dagger, a_2, a_2^\dagger, \dots, a_N, a_N^\dagger)^T.
\end{equation}
the Hamiltonian can be written as $H = \frac{1}{2} (\bm{\Psi}')^\dagger H' \bm{\Psi}'$, where $H'$ is a real, antisymmetric matrix. Any such matrix can be brought to a canonical block-diagonal form by a real orthogonal (special unitary) transformation $U$:
\begin{equation}
    U^T H' U = D = \bigoplus_{k=1}^N \begin{pmatrix} 0 & \epsilon_k \\ -\epsilon_k & 0 \end{pmatrix}, \quad \epsilon_k \ge 0.
\end{equation}
This transformation can be found using standard numerical libraries (e.g., via a real Schur decomposition). The new vector of operators $\tilde{\bm{\Psi}} = U^T \bm{\Psi}'$ will have components $(b_1, b_1^\dagger, b_2, b_2^\dagger, \dots)^T$, where $b_k, b_k^\dagger$ are the desired Bogoliubov fermion operators. The Hamiltonian is automatically diagonal in this basis:
\begin{equation}
    H = \frac{1}{2} \tilde{\bm{\Psi}}^\dagger D \tilde{\bm{\Psi}} = \sum_k \epsilon_k \left(b_k^\dagger b_k - \frac{1}{2}\right).
\end{equation}
The columns of the matrix $U$ directly provide the linear combination of the original $a_n, a_n^\dagger$ that form the new $b_k, b_k^\dagger$.

\subsection{Mode energy and occupation}






To express the mode occupation operators $\gamma_k^\dag \gamma_k$ in terms of the original fermionic operators, we use the Bogoliubov coefficients derived from the eigenvectors:
\begin{align}
    u_{nk} &= \frac{\mathcal{N}_k}{2}[\sin n\omega_k + \sin(N+1-n)\omega_k]\\
    v_{nk} &= \frac{\mathcal{N}_k}{2}[\sin(N+1-n)\omega_k - \sin n\omega_k]
\end{align}
The mode occupation is given by:
\begin{align}
    \gamma_k^\dag \gamma_k &= \sum_{n,m=1}^N \Bigl[u_{nk}^* u_{mk} a_n^\dag a_m + u_{nk}^* v_{mk} a_n^\dag a_m^\dag\\
                           &\quad+ v_{nk}^* u_{mk} a_n a_m + v_{nk}^* v_{mk} a_n a_m^\dag \Bigr]
\end{align}
Using the anticommutation relation $a_n a_m^\dag = \delta_{nm} - a_m^\dag a_n$, and applying the JW transformation from \cref{eq:JW_a_n,eq:JW_a_n_dag}, we can express $\gamma_k^\dag \gamma_k$ in terms of spin operators:
\begin{widetext}
    \color{red}
    \begin{align}
        \gamma_k^\dag \gamma_k &= \sum_{n,m=1}^N \left[u_{nk}^* u_{mk} \left(S_n S_m \frac{\sigma_n^+ \sigma_m^-}{4}\right) + u_{nk}^* v_{mk} \left(S_n S_m \frac{\sigma_n^+ \sigma_m^+}{4}\right) \right. \notag\\
                               &\quad \left. + v_{nk}^* u_{mk} \left(S_n S_m \frac{\sigma_n^- \sigma_m^-}{4}\right) - v_{nk}^* v_{mk} \left(S_n S_m \frac{\sigma_m^+ \sigma_n^-}{4}\right)\right] + \sum_{n=1}^N v_{nk}^* v_{nk}
    \end{align}
\end{widetext}
where $\sigma_n^{\pm} = \sigma_{n,x} \pm i\sigma_{n,y}$ are the spin raising and lowering operators, and $S_n S_m$ represents string operators:
\begin{align}
    S_n S_m =
    \begin{cases}
        \prod_{j=\min(n,m)}^{\max(n,m)-1} \sigma_{j,z} & \text{if } n \neq m \\
        1                                              & \text{if } n = m
    \end{cases}
\end{align}
The mode energy $E_k = \epsilon_k (2\gamma_k^\dag \gamma_k - 1)$ can thus be measured through these correlation functions of Pauli operators.
